# Newsfeedreader

First of all, you need to set up correct environment variable for MONGO_URL.
For example ```127.0.0.1:27017```.

## Get the project
Run ```go get github.com/jelinden/newsfeedreader```

## Build the project
```go build```

## Running the project
```go build && bash minify.sh && ./newsfeedreader```
